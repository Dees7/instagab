Instagrab
=========

Скачивает все фото и видео из Instagram, раскладывает по папкам - имя_пользоателя\

если в папке есть файл skip - у этого пользователя скачивается только фото с первой страницы (первые 19 шт)

Требования
----------
Для работы нужен jq 

использование
------------
в `users.txt` список учеток запустить `instagrab.sh`
